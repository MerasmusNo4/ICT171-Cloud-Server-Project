## Project Summary:

The application to be developed as part of this project is called "AI Central". The purpose of AI Central is to allow users to create their own customisable library of existing AI software, granting them quick and easy access to a wide range of AI tools and language models. The application will also include its own AI tools which can evaluate different AI models and detect AI generated content.

### Project Cost:
Two main components of this project required/ may require payment:

1. The DNS - The server's domain name was purchased for a low price of $7.98 p.a. from NameCheap
2. IaaS - For this project, use of the virtual machine on Microsoft Azure was purchased with free student credits. The price for a VM with the same configurations is an estimated $1,095.00 p.a. according to Azure's cost calculator.
