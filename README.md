## Project Summary:

The application to be developed as part of this project is called "AI Central". The purpose of AI Central is to allow users to create their own customisable library of existing AI software, granting them quick and easy access to a wide range of AI tools and language models. The application will also include its own AI tools which can evaluate different AI models and detect AI generated content.
