# Wordpress

So far, the website can only display a singular static page with basic HTML code. This can be improved without spending many hours writing line after line of HTML code by installing the appropriate sotware. 

This stage of the project covers the installation and configuration of Wordpress on the cloud server. 

Wordpress will be used to host all the static content of the cloud server (e.g. company info, marketing, contact info, development blog) throughout the development of the project.
